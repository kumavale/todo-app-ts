# Todoアプリ

Reactアプリに入門した with TypeScript and Rust

## 仕様

- 完了／未完了別にリスト化して表示(Read)
- 追加ができる(Create)
- 状態（完了／未完了）をボタンクリックで変更できる(Update)
- 削除ができる(Delete)

## 実行

1. バックエンドの起動

```
$ cd backend
$ cargo run
```

2. Reactアプリの起動

```
$ cd client
$ npm start
```

3. ブラウザから [http://localhost:3000](http://localhost:3000) へアクセス
